# 깨달은 점 
- 파라미터가 필요한 함수에 파라미터를 안넣으면 당연히 값이 안나온다.
- 웹 요소들은 비동기로 로딩되기 때문에 로드 순서가 중요하다.
- 움직임을 제어하는 요소들은 함수로 감싸주어야 인식이 가능하다. $(function(){});
- jQuery는 파일로 직접 넣어주어도 되고 링크를 걸어도 된다. (버전 잘맞추기)
- jQuery 사용시 문 잘 닫기 (;)
